Rubikube - A Rubik's Cube Game
Obs! First of all be sure the focus is on the Rubik's cube.
using letters
face rotation clockwise: l, r, u, d, f, b
face rotation counter-clockwise: shift + (l, r, u, d, f, b)
cube rotation: left, right, up, down arrows
using numeric pad
face rotation clockwise: 4, 6, 8, 2, 5, 9
face rotation counter-clockwise: shift + (4, 6, 8, 2, 5, 9)
cube rotation: ctrl + (4, 6, 8, 2)
using mouse
face rotation clockwise: click the center cell
face rotation counter-clockwise: shift + (click the center cell) or
right click the center cell
cube rotation: none
